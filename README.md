A tener en cuenta: Al trabajar con Java Swing e implementar el dibujo con coordenadas,
se tiene que en sistemas operativos con pantallas escaladas distintas a 100%, los elementos
se desfasan. Es por eso que se recomienda cambiar la escala a 100% en caso de no tener tal
configuración para una correcta visualización de la interfaz gráfica de nuestro proyecto.
